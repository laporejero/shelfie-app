# shelfie-app
A simple mobile app built with React Native (Expo) and Appwrite that allows users to manage their personal book collection. Users can create, view, edit, and delete books, with real-time synchronization across the app.

Features
- User authentication (via Appwrite)
- Create new books with title, author, and description
- View detailed book information
- Edit existing books
- Delete books
- Real-time updates using Appwrite subscriptions
- Light and dark mode support
- Mobile-first UI built with reusable themed components

Tech Stack
- React Native (Expo)
- Expo Router (file-based navigation)
- Appwrite (backend-as-a-service)
- Custom Themed Components (UI consistency)

Project Structure
- /app – Screens and routes (Expo Router)
- /components – Reusable UI components
- /context – Books state management (CRUD + realtime sync)
- /hooks – Custom hooks for data and auth
- /lib – Appwrite configuration

What I Learned
- Building full CRUD operations in a mobile app
- Integrating Appwrite as a backend service
- Handling real-time data updates
- Structuring React Native apps using Expo Router
- Creating reusable themed components for scalable UI design
